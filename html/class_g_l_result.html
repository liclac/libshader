Detailed Description

template<void(*)(GLuint, GLenum, GLint *) ivfunc, void(*)(GLuint, GLsizei, GLsizei *, GLchar *) logfunc, GLenum operation>
class GLResult< ivfunc, logfunc, operation >

A Result that can automatically query an OpenGL object for data. You should probably not use this directly, but rather typedef standard operation results to be able to use them more easily.

Template Parameters
ivfuncThe glGet___iv function used to query the object (eg. glGetShaderiv)
logfuncThe glGet___InfoLog function used to retrieve a log (eg. glGetShaderInfoLog)
operationThe operation to query (eg. GL_COMPILE_STATUS)
